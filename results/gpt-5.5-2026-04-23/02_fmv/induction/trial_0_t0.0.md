   If the effort stays unchanged, the pace stays unchanged. The object does not gather speed over time.

3. **There is no start-up period.**
   When a steady push or pull begins, the object immediately moves at the pace belonging to that effort. It does not pass through slower paces first.

4. **Pace is directly matched to effort.**
   Twice the effort gives twice the pace. Three times the effort gives three times the pace. If the same object is pulled the same way, the pace is simply proportional to how hard it is pushed or pulled.

5. **Heaviness reduces pace in the same proportion.**
   For the same steady effort, a block twice as heavy moves at half the pace. A block three times as heavy would move at one third the pace. Thus the pace depends on the effort divided by how heavy the object is.

6. **The track is not what causes the stopping.**
   The same rules hold on a level track and in open space. Touching the track is not needed for the object to halt when the push or pull ends.

7. **Travel requires continuous effort.**
   To move a load from one place to another, the carrier must keep pushing or pulling the whole way. A strong push at the beginning only moves the load while that push is present; after that, the load does not carry itself onward.

8. **Efforts in the same direction add.**
   If two carriers push the same block side by side in the same direction, the block’s pace is the sum of the paces that the two efforts would have produced separately.

9. **Opposed efforts subtract.**
   If two carriers push equally in opposite directions, their efforts balance and the block stays put. If one pushes harder, the block moves in the stronger direction, with a pace matching only the leftover difference between the two efforts.

10. **Falling is caused by a constant downward pull.**
   Released objects are pulled downward by the world with a steady effort. Because the effort is steady, the fall has one steady, unchanging pace.

11. **The downward falling pace is the same for heavy and light things.**
   The world’s downward pull on an object grows in step with the object’s heaviness. Since greater heaviness also makes a given effort produce a slower pace, the two effects cancel. A boulder and a pebble therefore fall side by side at the same steady pace.

12. **Letting go removes the hand’s forward push.**
   A stone carried forward by a hand moves forward only because the hand is carrying and pushing it along. Once released, that forward push is gone, so the stone has no reason to continue forward. It then responds only to the steady downward pull and drops straight down from the release point.

13. **Air does not set the falling pace.**
   Objects fall at the same steady pace in open air and in a jar with the air removed. The falling rule belongs to the object and the world’s downward pull, not to the presence or absence of air.

14. **The general rule of pace is: leftover effort decides motion.**
   Add all pushes and pulls in each direction. Opposite directions cancel against one another. If no leftover effort remains, the object stays put. If some leftover effort remains, the object moves at once in that direction at a steady pace set by the size of the leftover effort and by how heavy the object is.

15. **The general rule of stopping is: no present effort, no motion.**
   Past pushing does not matter. Only the push or pull acting now determines whether the object moves and how fast it moves.

No listed observation remains in tension with these rules. The same pattern explains pulling on a track, pushing in open space, combined carriers, opposed carriers, falling bodies, and the released stone.
