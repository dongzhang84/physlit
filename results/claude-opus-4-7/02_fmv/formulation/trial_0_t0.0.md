# Operational Rules

1. **Effort produces pace directly.**
   - *Operational form:* For any object under a steady effort E, the object moves at a steady pace P from the instant the effort begins until the instant it ends. P takes its full value immediately; no in-between paces occur at the start or finish.
   - *Scope:* Any object subject to one or more steady efforts, on a track or adrift, in any direction the efforts allow.
   - *What changes / stays the same:* The object's position changes at the rate P while E lasts. P itself stays the same throughout the interval of steady E. Nothing carries over once E ceases.
   - *Boundary cases:* Efforts that vary in time are not addressed; one must treat each interval of steady effort separately.

2. **Pace equals effort divided by heaviness.**
   - *Operational form:* For an object of heaviness H under a single steady effort E, the resulting pace is P = E ÷ H. Doubling E doubles P; doubling H halves P. To predict pace, divide the applied effort by the object's heaviness.
   - *Scope:* Any single object with a definite heaviness under a single net effort (after combining efforts by rule 4).
   - *What changes / stays the same:* While E and H are fixed, P is fixed. H is a property of the object and does not change with motion. P changes only if E changes.
   - *Boundary cases:* The rule requires a definite value of "heaviness" for the object; observations do not specify how heaviness is measured independently of comparing paces under equal efforts.

3. **Zero effort gives zero pace.**
   - *Operational form:* If the net effort on an object is zero (no efforts present, or efforts canceling per rule 4), then P = 0: the object remains at its current position indefinitely.
   - *Scope:* Any object, anywhere, at any moment when net effort is zero, including the instant after a push is released.
   - *What changes / stays the same:* Position stays the same. No prior motion is retained.
   - *Boundary cases:* This is the rule that forbids any lingering motion after release; it must be applied even at the very instant of release (see rule 8).

4. **Efforts combine by signed addition along a direction.**
   - *Operational form:* Assign each effort a sign according to its direction along a chosen line (+ one way, − the other). The net effort is the algebraic sum E_net = E₁ + E₂ + …. Then P = E_net ÷ H, with sign indicating direction. If E_net = 0, P = 0. For efforts not along a single line, combine direction by direction; the observations only directly address same-line cases.
   - *Scope:* Any object under two or more simultaneous steady efforts.
   - *What changes / stays the same:* E_net and therefore P stay the same as long as each contributing effort stays the same. If any one effort changes or ceases, recompute E_net and P immediately.
   - *Boundary cases:* The observations cover same-direction and opposite-direction combinations. Combinations at an angle are not directly attested; the rule as stated extrapolates by treating each direction independently.

5. **The surroundings do not enter the rule.**
   - *Operational form:* When computing P = E_net ÷ H, no term referring to the track, the medium, or the location appears. The same E and H give the same P whether the object is on a track or adrift.
   - *Scope:* All situations covered by rules 1–4.
   - *What changes / stays the same:* The predicted pace stays the same across changes of surroundings, given the same efforts and heaviness.
   - *Boundary cases:* None within the observations; this rule simply forbids adding surrounding-dependent terms.

6. **Falling is pace from a downward pull proportional to heaviness.**
   - *Operational form:* An unsupported object experiences a downward effort E_fall = k · H, where k is one fixed number, the same for every object. Its falling pace is therefore P_fall = E_fall ÷ H = k. Every unsupported object descends at the single pace k, beginning the instant support is removed and continuing unchanged until support is restored or another effort intervenes.
   - *Scope:* Any object not supported and not held by another effort that cancels E_fall.
   - *What changes / stays the same:* The downward position changes at rate k. The pace k itself does not change during the fall and does not depend on which object is falling.
   - *Boundary cases:* If an upward effort acts as well, combine by rule 4: net downward effort is k·H − E_up, giving downward pace k − E_up/H. If E_up = k·H, the object hovers.

7. **The downward pull is present regardless of medium.**
   - *Operational form:* E_fall = k · H applies whether or not air surrounds the object. No term for the medium appears in the falling rule.
   - *Scope:* Any unsupported object, in air, in a void, or in any surrounding the observations consider.
   - *What changes / stays the same:* P_fall = k is the same in all such surroundings.
   - *Boundary cases:* Surroundings that exert their own effort on the object (e.g., a wind, a current) would have to be entered as additional efforts under rule 4; the observations do not treat such cases.

8. **A held object takes its carrier's pace only during contact.**
   - *Operational form:* While a carrier applies effort E_side sideways to a held object of heaviness H, the object's sideways pace is P_side = E_side ÷ H, and its downward pace is governed by rule 6 only insofar as it is unsupported in that direction (if the hand also supports it vertically, the supporting effort cancels E_fall and downward pace is zero). At the instant of release, all efforts from the carrier cease; by rule 3, sideways pace becomes 0 immediately, and by rule 6, downward pace becomes k immediately. The object thereafter descends straight down from the point of release.
   - *Scope:* Any object in contact with a carrier that applies effort, and the same object just after release.
   - *What changes / stays the same:* Sideways pace changes from P_side to 0 at the instant of release; downward pace changes from 0 (if supported) to k at the same instant. Nothing of the prior sideways motion is preserved.
   - *Boundary cases:* The rule requires the transition at release to be instantaneous, with no residual sideways pace. This is the most delicate point flagged in Stage 1.
