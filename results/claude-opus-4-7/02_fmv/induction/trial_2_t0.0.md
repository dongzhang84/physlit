# Rules of this world

1. **Effort and pace are directly linked, at the same instant.** Whenever a push or pull is being applied to an object, the object moves; the moment such effort ceases, the object is at rest. There is no carry-over: nothing about the past push lingers in the object.

2. **The pace is set by the effort, not built up over time.** As soon as a push begins, the object is already moving at its full steady pace. As long as the push continues unchanged, the pace stays unchanged. The pace does not grow with the duration of the push.

3. **Pace grows in proportion to effort.** Double the steady effort gives double the steady pace; triple the effort, triple the pace. Halving the effort halves the pace.

4. **Heavier objects yield less pace for the same effort.** For a given steady push, a block twice as heavy moves at half the pace; three times as heavy, one third the pace. The pace an object takes on equals the effort applied to it divided by how heavy it is.

5. **Pushes acting on the same object combine by direction.** Two pushes in the same direction produce a pace equal to the sum of the paces each would produce alone. Two pushes in opposite directions produce a pace set by the difference of the efforts, in the direction of the stronger one; equal opposing efforts leave the object at rest.

6. **These rules do not depend on the track or on the surroundings.** A block adrift in open space, touching nothing, follows the same rules as a block on the track: steady push gives steady pace, release gives immediate halt.

7. **Carrying requires continuous pushing.** Because nothing about a past push lingers, a load left to itself stops at once. To move a load from here to there, the carrier must push it the whole way; an early hard shove followed by release leaves the load standing where the shove ended.

8. **A held object shares the carrier's motion only while held.** While a stone is in the hand, the hand's push moves it along. The instant the stone is released, no push acts on it sideways, so it has no sideways pace; it only falls.

9. **Falling has its own fixed pace, the same for every object.** When an object is released above the ground, it descends at one single steady downward pace that does not grow during the fall. This pace is the same for a boulder and for a pebble, and it is the same whether the air is present or has been removed. (In the language of rule 4, the downward pull on an object is exactly matched to how heavy that object is, so the resulting downward pace comes out the same for everything.)

10. **The fall halts on contact.** When the falling object reaches the ground, the downward pull is opposed and the object stops, in keeping with rule 5.

# Tensions

- **Observation 11** fits the rules only if we also accept rule 8 — that a carried object takes on the carrier's sideways pace only while contact and push are maintained. This is consistent with rule 1 (no lingering effect of a past push), but it does require us to say that, during carrying, the hand is continuously imparting the sideways pace to the stone. That is a reasonable reading, but the observation does not directly confirm it; it only shows the after-release behavior.

- **Observation 12** sits awkwardly with rule 4. By rule 4, a heavier object should need a proportionally stronger pull to keep pace with a lighter one — and indeed it does, for the fall pace to come out equal for boulder and pebble, the downward pull on each object must be exactly tuned to how heavy that object is. The rules accommodate this, but only by treating the downward pull as a special kind of effort that scales with heaviness, unlike the hand-pulls of observations 4 and 5.
