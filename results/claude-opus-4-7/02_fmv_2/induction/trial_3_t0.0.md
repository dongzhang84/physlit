# Rules of this world

1. **An object moves only while it is being pushed or pulled.** The moment any push or pull begins, the object is already gliding; the moment it ends, the object halts on the spot. There is no gradual speeding-up and no drifting onward.

2. **The pace of an object is set, at every instant, by the effort applied to it, divided by how heavy it is.** Double the effort gives double the pace; double the heaviness gives half the pace. Pace is fixed by the present push, not built up over time.

3. **When several pushes or pulls act on one object, only their combined leaning matters.** Pushes in the same direction add together; pushes in opposite directions subtract. The object's pace and direction follow this single net effort by Rule 2. If the net effort is zero, the object stays put.

4. **Falling objects are pulled downward by a steady downward effort whose size is exactly proportional to how heavy the object is.** By Rule 2, the resulting downward pace is therefore the same for every object — a boulder and a pebble fall together at one unchanging pace — and this pull does not depend on whether air is present.

5. **Being carried by something else does not impart any push to an object.** Once released, an object has only whatever pushes or pulls currently act on it directly; previous handling leaves no trace. A released stone therefore simply falls straight down under the pull of Rule 4.
