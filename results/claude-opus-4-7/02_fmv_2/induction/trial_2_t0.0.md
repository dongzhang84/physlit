# Rules for this world

1. **An object moves only while it is being pushed or pulled.** The instant any push or pull is applied, the object moves; the instant it stops, the object halts where it is. There is no carrying-onward and no gradual speeding-up: the pace at any moment is whatever the current pushing sets it to.

2. **The pace of an object equals the total effort applied to it, divided by its heaviness.** "Total effort" means the combined effort of all pushes and pulls, with opposing pushes subtracting and same-direction pushes adding. If the total is zero, the object stays put; if the total points one way, the object moves that way at the pace fixed by this rule.

3. **Every object, simply by being an object, is pulled steadily downward.** The downward pull on an object is exactly proportional to its heaviness, so by rule 2 every released object falls at the same single steady pace, regardless of how heavy it is and regardless of what (if anything) surrounds it. A released object's sideways travel ends at once (rule 1), so it drops straight down from the point of release.

4. **The track, the open air, and empty space all behave alike.** None of them pushes an object on their own and none of them keeps an object going. The rules above hold the same way in every setting.
