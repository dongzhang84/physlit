**1. Push-tied motion.**
- *Operational form:* At every instant t, an object's pace P(t) is nonzero if and only if a nonzero push acts on it at that instant. P(t) is determined entirely by the push present at time t. There is no contribution from any push that acted at any earlier time.

- *What changes / stays the same:* The pace changes the instant the push changes; it takes whatever value the current push dictates. Nothing about the prior motion is carried forward into the next instant. No quantity is preserved across a change of push.
- *Boundary cases:* The rule demands that pace switch on and off perfectly with the push, with no transitional interval. Observation 11 (release of a carried stone) sits at this boundary: the sideways push must vanish exactly at the moment of release.

**2. Pace from push and heaviness.**
- *Operational form:* For a single push of size F acting on an object of heaviness H, the pace is P = F / H, in the direction of the push. Doubling F at fixed H doubles P; doubling H at fixed F halves P. Given any F and H, a third party computes P = F/H directly.
- *Scope:* Any object under a single push or pull, including the downward pull of Rule 4. For multiple pushes, first combine them by Rule 3, then apply this rule to the combined push.
- *What changes / stays the same:* H is a fixed property of the object and does not change as it moves. F may change from instant to instant; P tracks F/H instantaneously. The ratio P/F is constant for a given object (equal to 1/H); nothing else is preserved.
- *Boundary cases:* The rule requires H > 0 and gives no prescription if F is applied to an object of unspecified heaviness.

**3. Combination of pushes.**
- *Operational form:* When pushes F₁, F₂, … act simultaneously on one object along a line, the combined push is their signed sum (same direction adds, opposite direction subtracts). For pushes not along one line, take the directional combination: line them up tail-to-head and use the resulting single arrow as F_net. Then P = F_net / H in the direction of F_net. If F_net = 0, then P = 0 and the object stays in place.
- *Scope:* Any object on which more than one push or pull acts at the same instant.
- *What changes / stays the same:* F_net may change at any instant; P follows it immediately by Rule 2. No quantity is preserved across changes in the set of pushes.
- *Boundary cases:* When opposing pushes exactly cancel, the object is motionless even though pushes are present — consistent with Rule 1 read as "nonzero net push" rather than "any push."

**4. Downward pull.**

By Rule 2, the downward contribution to pace is F_down / H = k, the same value for every object regardless of heaviness. The presence or absence of air does not alter F_down or k.
- *Scope:* All objects, at all times, in any surroundings. This pull is one of the pushes entering Rule 3.
- *What changes / stays the same:* k is the same for all objects and does not vary in time or place within the observations given. F_down for a particular object is fixed (since H is fixed). The downward component of pace contributed by this pull is the constant k.
- *Boundary cases:* None among the listed observations; (10) and (12) follow directly.
